# Slidev Presentation

This repository contains a Slidev presentation. Follow the instructions below to get started.

## Prerequisites

Before you begin, ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your system.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-folder>
    ```

2.  **Install dependencies:**
    Navigate to the `260205` directory and install the necessary packages.
    ```bash
    cd 260205
    npm install
    ```

3.  **Run the presentation:**
    To start the Slidev presentation in development mode, run the following command:
    ```bash
    npm run dev
    ```
    This will open the presentation in your browser.

### Other available commands

-   **Build for production:**
    ```bash
    npm run build
    ```

-   **Export as PDF:**
    ```bash
    npm run export
    ```

## Recommended VS Code Extensions

To enhance your experience working with this Slidev presentation, we recommend installing the following VS Code extensions:

-   [**Slidev**](https://marketplace.visualstudio.com/items?itemName=antfu.slidev) (`antfu.slidev`): The official extension for Slidev. It provides a seamless experience for creating and managing your presentations with features like a live preview, slide navigation, and more.

-   [**GitHub Copilot**](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) (`GitHub.copilot`): Your AI-powered assistant for everything, including writing Slidev presentations. It can help you generate slide content, write code snippets, and much more, directly within the editor.
